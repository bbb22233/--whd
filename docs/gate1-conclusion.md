## 0. 一句话结论
**雷达能"诚实地描述当下是什么环境",但没有任何回测证据表明它能转化成"可交易的钱"。**
它的定位是**环境/波动描述器 + 风控背景层**,**不是预测器、不是 alpha 源**。在它上面加杠杆没有依据。

## 1. A 层(状态本身有没有区分力)
- **方向命中 ≈ 基准率**(M2 已暗示约 45–46%):雷达**不预测涨跌方向**。
- **波动环境有区分力**:RANGE 状态显示出对未来的正 lift(1D +5 / 4H +13 / 8H +12 量级);CHAOS 是**当下高波动的描述**,不是未来预测。
- 结论:**作为"现在是什么天气"的描述,雷达是靠谱、无过拟合的**(M2 状态稳定性审计 + Gate-1 共同支持);作为"接下来涨还是跌"的预测器,**无 edge**。

## 2. B 层(状态当过滤层有没有用)——三组都没立住
| 组 | 设计 | 结论 |
|---|---|---|
| **S1 RANGE 收割** | 均值回归基准,只在 RANGE 灯亮时开仓 | **不成立(测试载体本身坏了)**。表面 ΔSharpe +0.6/+0.8,实为**假象**:暴露率仅 0.45–0.94%、合格品种仅 1–10 个。根因——均值回归入场要"价偏离 ≥1.5 ATR 的拉伸",而 RANGE 恰恰意味着"平静、无拉伸",**两者几乎从不同时出现**,于是过滤版约等于"不交易一个本会爆仓(基准 98% MaxDD)的破策略"。+ΔSharpe 是"没去做烂交易",不是"RANGE 有 edge"。**RANGE 收割:悬而未决,需换一个不自相矛盾的载体才能公平重测。** |
| **S2 躁动规避(避 CHAOS)** | 时序动量基准,CHAOS 灯亮时减仓/空仓 | **≈ 中性**。没明显改善夏普,尾部也没看到稳定收益。 |
| **S3 TREND 过滤(诚实对照)** | 动量只在 TREND 灯亮时做 | **确认有害**:ΔSharpe 全为负。**坐实"别用方向那条"——一个干净的负结果。** |

## 3. 综合裁决
- **TREND 方向:确认无用**(S3 变差)。
- **CHAOS 规避:中性**(S2 不亏不赚)。
- **RANGE 收割:悬而未决**(S1 测试载体自相矛盾,非真信号也非真无效)。
- **总:无任何已证实的可交易 edge。** 雷达的价值在于诚实地告诉你"现在是什么环境",供**监控 / 风控 / 仓位语境**参考,**不作为下单信号**。

## 4. 收手后的定位(给前端/产品的话术约束)
- 对外措辞只能是 **"环境/状态描述 + 概率语境"**,**禁止**任何"该买/该卖/预测涨跌"的暗示。
- 概率旁必须标**样本数 / 置信**(对应未做的 N4/L5),防止把"描述"误读成"确定性"。
- 任何"加杠杆""跟单"类用法**无回测背书**,不得据此宣传。

## 5. 若将来想再追(不在本轮)
- 唯一值得给一次干净机会的是 **RANGE 收割**:换一个**与 RANGE 不矛盾**的载体(例如区间内的网格/边界回归,而非"等大拉伸"的均值回归),并这次**提交小结果文件**留痕。
- 任何正式结论需补 **Deflated Sharpe / PBO**(López de Prado)做多重检验校正(原规划的 G1-C)。
- 在那之前,**不在没有 edge 的东西上加杠杆**——这是 Gate-1 最大的价值。
