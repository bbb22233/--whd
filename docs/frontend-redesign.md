# 前端重塑:React 数据密集型监控终端(frontend-craft)

> 依据技能 `skills/frontend-craft/`(第五章 industrial/量化终端 + 第三章 AI 信任校准 + 第二章设计思维)。
> 旧原生前端(`app.js`/`index.html`/`styles.css`)已删除;前端主线只保留 `web/`。

## 设计

**场景类别:数据密集型监控终端**(主)+ **AI 信任校准**(叠加,因雷达有样本/置信且定位"描述非预测")。
**风格方向:industrial / 量化终端**——信息密度第一,装饰最后。

### 布局架构
```
┌─ Topbar ───────────────────────────────────────────────┐
│ 品牌  [品种▾][周期▾]  样本/区间/源 chips      ☀ ↻      │
├──────────────────┬──────────────────────────────────────┤
│ 总闸门 GatePanel  │ 盘口快照 + 1px 价格线(recharts)     │
│  灯号●+路由+置信  ├──────────────────────────────────────┤
│  小样本弱化+红线  │ (left)            (right)            │
│ 组件状态卡 ×6     │ 概率拆解表(置信/样本)               │
│ 指标工厂(密表格) │ 乖离规则表 / 当前读法                 │
└──────────────────┴──────────────────────────────────────┘
         footer:routerPrinciple · 描述环境,非买卖信号
```

### 从技能取了什么
- **第五章(占比最高)**:暗色去饱和石板 token(`--bg-base:#0B0E14` 全套)+ 亮色主题;所有数字 **JetBrains Mono + tabular-nums + 右对齐**;1px 边框、0-3px 圆角、平且安静;**颜色 + 符号双编码**(灯号 ●◐◑○、涨跌 ▲▼);**唯一动效 = 变值闪烁**(`FlashCell`,250ms 背景涨绿/跌红,不改布局);密表格 + 分割面板 > 卡片网格;1px 价格线、无面积填充。
- **第三章(信任校准)**:置信不用裸百分比,转 **High/Medium/Low + 数值**;**样本数**显式展示;小样本(occ<30 或样本闸非"通过")**整卡弱化** + ⚠ 警示;每屏底部"描述环境,非买卖信号"。
- **第二章**:差异化锚点——**对不确定性诚实的"天气雷达"**,把信任校准当设计特色而非免责声明。

### 关键设计决策
- **配色**:石板暗底 + 金融语义色(涨 `#2EBD85` / 跌 `#F6465D` / 警告 `#F0A020` / 强调 `#38BDF8`),灯号映射 绿=up、黄=warn、黄偏绿=up-soft、红=down。
- **字体**:UI 用 Inter,**所有数字与符号用 JetBrains Mono**(等宽不抖)。
- **主题**:暗/亮一键切换(localStorage 记忆),`[data-theme]` 驱动。

## 技术栈与数据
- **React 19 + Vite 7 + recharts 3**;`base:"./"` 产物可同源/任意路径部署。
- **数据层复用现有 REST 契约**:主路 `/api/dashboard/current`,失败回退静态 `/reports`、`/data`(`server.mjs` 提供)。无新增后端接口、不碰研究算法。
- **保留的行为**:M5 样本不足占位、L5 样本/置信、M4 `ma233` 规则、L2 由 React 默认转义天然满足。
- **红线**:全程"只描述环境/状态,不预测涨跌、不给买卖点"。

## 交互功能
- 切品种/周期 → 更新 URL(`?instrument=&bar=`)+ 重渲染;浏览器前进后退(popstate)跟随。
- API 挂掉自动回退静态文件(按所选品种)。
- 刷新按钮、主题切换。
- 薄历史品种 → "样本不足"占位屏,不崩。
- 数据刷新时盘口价格变值闪烁。

## 运行
```bash
# 开发(配合 FastAPI:8000,vite 代理 /api)
cd web && npm install && npm run dev        # http://127.0.0.1:5173
# 生产构建 → web/dist,由 server.mjs 服务
npm run build && cd .. && node server.mjs   # http://127.0.0.1:4177
```
